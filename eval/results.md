# Evaluation Results

This file is generated by:

```bash
PYTHONPATH=. uv run python eval/run_eval.py
```

The committed result should not be treated as a permanent quality claim. Run the evaluation after changing prompts, retrieval, routing, model configuration, the knowledge base, or `RETRIEVAL_MAX_DISTANCE`.

The generated report includes:

- timestamp;
- commit SHA when available through CI;
- retrieval threshold;
- per-model pass counts;
- hallucination count;
- runtime error count.
